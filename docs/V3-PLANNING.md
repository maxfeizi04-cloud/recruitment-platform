## 三、P0：核心体验补完（预计 3 天）

### 3.1 密码登录
**现状**: 只有验证码登录，Mock SMS 只能 debug 用
**方案**: 登录页增加"密码登录" Tab，注册时设密码（bcrypt 存储）

### 3.2 管理员审核功能页
**现状**: 管理员仪表盘只有 Mock 数据
**方案**:
- 企业认证审核列表（通过/驳回）
- 职位审核列表
- 用户举报处理

### 3.3 占位页面补完
**现状**: 收藏/关注/求职服务/职业测评/Offer管理/内容管理等页面都是空白
**方案**:
- 收藏职位页（展示收藏列表）
- Offer 管理页（HR 发 Offer → 求职者确认）
- 数据统计页（接入真实 API 数据，替代 Mock）

## 四、P1：交互体验提升（预计 2 天）

### 4.1 聊天界面
**现状**: IM UserSig API 已就绪，但没有聊天 UI
**方案**: 在面试邀请旁增加"联系 HR"按钮 → 打开聊天弹窗（腾讯云 IM SDK）

### 4.2 响应式移动端
**现状**: 仪表盘在大屏 OK，手机端布局破碎
**方案**: 全局检查 `md:` / `lg:` 断点，侧边栏移动端折叠，卡片移动端全宽

### 4.3 Toast 通知优化
**现状**: WebSocket 通知已通，但只是 message.info
**方案**: 通知中心 + Bell 红点数字 + 点击查看通知列表
